# API documentation

Этот файл содержит ссылки на внешнюю документацию по API, которую AI агенты могут использовать при работе с контрактами приложения.

## Ссылки

- [Alor OpenAPI HTTP overview](https://alor.dev/docs/api/http/overview)
- [Alor OpenAPI WebSocket API overview](https://alor.dev/docs/api/websocket/overview)
- [Alor OpenAPI GraphQL API overview](https://alor.dev/docs/api/graphql/overview)

## Правила для AI агентов

- Используй ссылки из этого файла, чтобы уточнять контракты API: endpoints, HTTP методы, параметры, DTO, структуры ответов, коды и форматы ошибок.
- При добавлении отсутствующих методов, сервисов или типов сверяй реализацию с актуальной внешней документацией из этого файла.
- Если задача затрагивает API-контракт, не полагайся только на похожий существующий код: проверь источник документации и сохрани совместимость с текущими архитектурными правилами репозитория.
